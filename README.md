## Pré-requisitos
* [docker](https://www.docker.com/)

* [GNU Make](https://www.gnu.org/software/make/)

## Guia para Uso
```sh
make docker  # builda e sobe o container
# or
make all     # desce, builda e sobe o container
# or
docker-compose up -d --build  # builda e sobe o container
```
